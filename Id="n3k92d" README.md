# Authentication System (Styled with Bootstrap)

## 📌 Project Description
This project is a responsive authentication system built using HTML, CSS, and Bootstrap.  
It includes Login, Register, Forgot Password, Reset Password, and Dashboard pages.

## 🚀 Features
- Responsive design (Mobile, Tablet, Desktop)
- Bootstrap styling
- Form validation UI
- Smooth UI with shadows and spacing

## 📂 Pages Included
1. Login Page (index.html)
2. Register Page (register.html)
3. Forgot Password Page (forgot-password.html)
4. Reset Password Page (reset-password.html)
5. Dashboard Page (dashboard.html)

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5
